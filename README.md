# 🛡️ Paygentic Fraud Detector

An AI-powered cybersecurity web application that detects suspicious payment transactions using smart fraud detection logic.

---

## 🚀 Features

- 🔍 Detects High, Medium, and Low Risk Transactions
- 🚨 Fraud Alert Alarm System
- 📊 Live Risk Distribution Chart
- 📁 Transaction History Tracking
- 📄 Download Fraud Report
- 🌙 Dark / Light Mode UI
- 📱 Responsive Cybersecurity Dashboard
- 📍 Location-based Fraud Detection
- ⏰ Late Night Transaction Detection

---

## 🧠 Fraud Detection Logic

The system checks:

- Transaction Amount
- Transaction Time
- Suspicious Locations
- Payment Method
- Risk Score Calculation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
